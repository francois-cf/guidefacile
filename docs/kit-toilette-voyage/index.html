Kit de toilette avion / voyage

Le plus simple : une trousse souple + flacons avion réutilisables. Budget 12–25 €.

Voir les derniers guides

Comment choisir

Vérifie taille (≤ 20x15 cm), flacons ≤ 100ml, matière lavable. Les modèles souples sont légers, les rigides protègent mieux.

À éviter

Les kits trop bas de gamme ont des bouchons qui fuient.
