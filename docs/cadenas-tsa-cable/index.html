Pourquoi choisir un cadenas TSA à câble ?

  • Accepté TSA : permet aux agents de sécurité d’ouvrir sans casser le cadenas lors des contrôles.
  • Câble flexible : plus pratique qu’une anse rigide pour attacher zips ou boucles.
  • Compact et léger : parfait pour valises, sacs à dos, sacs de sport.
  • Code personnalisable : sans clé, vous définissez votre combinaison.
  • Matériaux robustes : corps en alliage + câble acier gainé pour plus de résistance.
