Pourquoi cet adaptateur voyage ?

Avec ses prises pour l’Europe, le Royaume-Uni, les États-Unis, l’Australie ou l’Asie, cet adaptateur tout-en-un vous évite d’emporter plusieurs chargeurs. Il est compact, doté de 3 ports USB-A et 1 USB-C, et intègre une double protection (fusible) pour votre sécurité.

Fonctions clés

  • Multi-régions : UE / UK / US / AU / Asie (Japon, Chine, Thaïlande...)
  • 3 ports USB-A + 1 port USB-C
  • Double fusible intégré pour éviter les surcharges
  • Compact : se glisse dans toute poche de bagage

Conseils d’utilisation

Ce modèle est idéal en avion, en train ou à l’étranger. Attention, ce n’est pas un convertisseur de tension—assurez-vous que vos appareils acceptent 100-240 V. Vérifiez toujours les tensions sur vos appareils avant utilisation.
